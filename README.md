#To Run Local

```bash
# Clone this project
$ git clone  https://github.com/morshidulrahman/carassingment-4-backend.git

# Access
$ cd carassingment-4-backend

# Install dependencies
$ npm i

# Run the project
$ npm run start:dev

# The server will initialize in the <http://localhost:5000>
```
